# Minimal Economics Research Website

This is a plain static website for GitHub Pages. It has no build step and no
framework dependency.

## Files

- `index.html` is the home page.
- `research.html` lists publications, working papers, and work in progress.
- `assets/styles.css` controls the visual design.
- `assets/picture_me.png` is the profile photo used on the home page.
- `assets/CV_Goldemann.pdf` is the CV linked from the site.

## Editing Content

Open the HTML files and replace the placeholder text such as `Your Name`,
`University Name`, paper titles, coauthors, and contact details. Keep the file
names unchanged unless you also update the links in the navigation.

To add your CV, place the PDF at:

```text
assets/CV_Goldemann.pdf
```

To add your photo, place the image at:

```text
assets/picture_me.png
```

## Publishing With GitHub Pages

1. Create a GitHub repository named `<github-username>.github.io`.
2. Push these files to the default branch of that repository.
3. In GitHub, open the repository settings and enable Pages from the default
   branch root if it is not enabled automatically.
4. Visit `https://<github-username>.github.io/`.
